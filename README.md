# HiraKata
This is a simple program designed to help practice Hiragana and Katakana

To run the .exe (Windows only):
Simply double click the .exe file

To run the Java version:
1. Download HiraKata.jar
2. Download either the .bat (Windows), .command (macOS), or .sh (Linux) and double click to run

NOTE: Make sure the .jar and the .bat/.command/.sh are in the same folder
